# What is GitHub?

## Definition

GitHub is a cloud-based platform that hosts Git repositories online. It allows developers to store, manage, collaborate, and share software projects using Git.

---

## Why is GitHub Used?

- Store projects online
- Backup source code
- Collaborate with developers
- Track project history
- Contribute to open-source projects
- Showcase projects to recruiters

---

## How GitHub Works

```
Developer
     │
     ▼
Local Computer
     │
     ▼
Git
     │
     ▼
GitHub Repository
     │
     ▼
Other Developers
```

Git tracks your project locally.

GitHub stores that project on the cloud.

---

## Main Features

### Repositories

A repository stores your project's files and history.

---

### Branches

Used to develop new features without affecting the main project.

---

### Pull Requests

Request to merge changes into another branch.

---

### Issues

Used to report bugs, request features, or manage tasks.

---

### Actions

Automate testing, building, and deployment.

---

### Releases

Publish stable versions of a project.

## Real-World Example

Suppose you create a Calculator application.

```
Laptop
   │
   ▼
Git
   │
   ▼
GitHub
```

Whenever you make changes:

1. Git records the changes.
2. GitHub stores them online.
3. Team members can download the latest version.

---

## Git vs GitHub

| Git | GitHub |
|------|---------|
| Version Control System | Cloud Hosting Platform |
| Installed on a computer | Runs on the internet |
| Tracks file changes | Hosts Git repositories |
| Works offline | Requires internet |

---

## Summary

Git is a tool that tracks changes in a project.

GitHub is a platform that stores Git repositories online and enables collaboration between developers.
