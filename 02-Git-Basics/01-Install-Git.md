# Install Git

## Introduction

Before using Git, it must be installed on your computer. Git is available for Windows, macOS, and Linux, making it one of the most widely supported development tools.

Once installed, Git allows you to initialize repositories, track changes, create commits, and collaborate with developers worldwide.

---

# System Requirements

Git is lightweight and works on almost every modern computer.

Supported Operating Systems:

- Windows
- macOS
- Linux

---

# Download Git

Official Website:

https://git-scm.com

Always download Git from the official website to ensure you receive the latest stable version.

---

# Installing Git on Windows

### Step 1

Visit the official Git website.

### Step 2

Download the latest Windows installer.

### Step 3

Run the installer.

### Step 4

Keep the default installation settings unless you have a specific requirement.

### Step 5

Click **Install**.

### Step 6

After installation, click **Finish**.

---

# Verify Installation

Open Command Prompt, PowerShell, or Git Bash and run:

```bash
git --version
```

Example Output:

```bash
git version 2.51.0
```

If the version number appears, Git has been installed successfully.

---

# Git Bash

Git Bash is a terminal included with Git for Windows.

It provides a Linux-like command-line environment where developers can execute Git commands.

Common uses:

- Running Git commands
- Navigating directories
- Managing repositories
- Executing shell commands

---

# Update Git

To check your installed version:

```bash
git --version
```

Download newer versions only from:

https://git-scm.com

---

# Common Installation Problems

### Git command not recognized

Cause:

Git is not installed correctly or PATH is not configured.

Solution:

- Restart the computer.
- Reinstall Git.
- Ensure "Add Git to PATH" is selected during installation.

---

### Old Git Version

Download the latest installer and upgrade Git.

---

# Best Practices

- Download Git only from the official website.
- Keep Git updated.
- Learn Git Bash commands.
- Verify installation before creating repositories.
